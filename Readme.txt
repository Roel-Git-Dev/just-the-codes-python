Description
A simple code for string balance checker written in python.
A String balance checker is a type of program where it checks if braces, 
brackets, parentheses, etc are equal and properly close.

Files needed
Is_String_Balance.py (this is the main app)
FunctionList.py (contains functions that the main app uses)

*The instruction here is for Visual Studio only*
*But it can also be run in VS code, you just need to setup a python environment*

To run
1.) Download or clone both the needed files.
2.) Put them in a folder.
3.) Right click that folder and choose Visual Studio. 
    or 
    Open Visual Studio and choose "open a local folder".
4.) Select the Is_String_Balance.py and set it as Startup Item.
5.) Run the program enjoy!
